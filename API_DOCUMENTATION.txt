MH MARBLES: FULL API DOCUMENTATION 
=========================================================
Overview:
- Base URL: http://localhost:8000
- GET Requests are largely PUBLIC (No Token Required).
- POST/PUT/PATCH/DELETE require an 'Authorization: Bearer <token>' header from Spring Boot (except Contact Form POST).
- IMPORTANT: All image/media fields returned in GET responses dynamically generate temporary AWS S3 Pre-signed URLs (valid for 1 hour) instead of exposing static storage paths.
- IMAGE UPLOADS: For creating/updating records with images, send a `multipart/form-data` request with the actual image File in the `image` (or `logo`) key. The backend automatically uploads to S3.

---------------------------------------------------------
1. COMPANIES API
---------------------------------------------------------
[GET] /api/companies/ 
Auth Required: No (Public)
Description: Retrieves all companies/brands.
Response (200 OK):
[
  {
    "id": 1,
    "name": "Marble Kings",
    "logo_key": "companies/xyz_img.jpg",
    "logo_url": "https://mhm-appcomp-s3-01.s3.amazonaws.com/companies/img.jpg?X-Amz-..."
  }
]

[POST] /api/companies/
Auth Required: Yes (JWT Bearer Token)
Content-Type: multipart/form-data
Description: Create a new company.
Request Body (Send as FormData):
- name: "Marble Kings" (Text) [REQUIRED]
- logo: File (Image) [OPTIONAL] -> Automatically uploaded to S3
Response (201 Created): Returns the generated JSON object.

[PUT] /api/companies/{id}/
Auth Required: Yes (JWT Bearer Token)
Content-Type: multipart/form-data
Description: Fully update an existing company (all fields required).
Request Body (Send as FormData):
- name: "Marble Kings" (Text) [REQUIRED]
- logo: File (Image) [OPTIONAL] -> Automatically uploaded to S3
Response (200 OK): Returns the updated JSON object.

[PATCH] /api/companies/{id}/ & [DELETE] /api/companies/{id}/
Auth Required: Yes

2. HERO CAROUSEL API
---------------------------------------------------------
[GET] /api/hero_carousel/
Auth Required: No (Public)
Response (200 OK):
[
  {
    "id": 1,
    "heading": "Premium Tiles",
    "subtext": "Best in class",
    "image_key": "hero_carousel/xyz.jpg",
    "image_url": "https://mhm-appcomp-s3-01.s3.amazonaws.com/hero_carousel/img.jpg?...",
    "cta_text": "Shop Now",
    "cta_link": "/shop",
    "order": 1
  }
]

[POST] /api/hero_carousel/
Auth Required: Yes
Content-Type: multipart/form-data
Description: Create a new carousel slide.
Request Body (Send as FormData):
- image: File (Image) [OPTIONAL] -> Automatically uploaded to S3
- heading: (Text) [OPTIONAL]
- subtext: (Text) [OPTIONAL]
- cta_text: (Text) [OPTIONAL]
- cta_link: (Text) [OPTIONAL]
- is_active: true/false (Boolean) [OPTIONAL, default: true]
- order: (Number) [OPTIONAL, default: 0]
Response (201 Created): Returns generated JSON object.

[PUT] /api/hero_carousel/{id}/
Auth Required: Yes
Content-Type: multipart/form-data
Description: Fully update an existing carousel slide.
Request Body (Send as FormData):
- image: File (Image) [OPTIONAL] -> Automatically uploaded to S3
- heading: (Text) [OPTIONAL]
- subtext: (Text) [OPTIONAL]
- cta_text: (Text) [OPTIONAL]
- cta_link: (Text) [OPTIONAL]
- is_active: true/false (Boolean) [OPTIONAL]
- order: (Number) [OPTIONAL]
Response (200 OK): Returns updated JSON object.

[PATCH] /api/hero_carousel/{id}/ & [DELETE] /api/hero_carousel/{id}/
Auth Required: Yes

5. PRODUCTS API (Cursor Paginated)
---------------------------------------------------------
[GET] /api/products/
Auth Required: No (Public)
Supports direct query params: ?company=1&category=2&color=Red&size=600x600&finish=Glossy
Response (200 OK):
{
  "next": "http://localhost:8000/api/products/?cursor=cj1Qcm...",
  "previous": null,
  "results": [
    {
      "id": 1,
      "name": "Classic Carrara",
      "company": 1,
      "category": 2,
      "size": "800x800",
      "color": "White",
      "finish": "Matt",
      "sku": "SKU-CAR-01",
      "image_key": "products/carrara.jpg",
      "image_url": "https://mhm-appcomp-s3-01.s3.amazonaws.com/products/carrara.jpg?..."
    }
  ]
}

[POST] /api/products/
Auth Required: Yes
Content-Type: multipart/form-data
Description: Create a new product.
Request Body (Send as FormData):
- name: "Classic Carrara" (Text) [REQUIRED]
- company: 1 (Number/FK) [REQUIRED]
- category: 2 (Number/FK) [REQUIRED]
- size: "800x800" (Text) [REQUIRED]
- color: "White" (Text) [REQUIRED]
- finish: "Matt" (Text) [REQUIRED]
- sku: "SKU-CAR-01" (Text) [REQUIRED, Unique]
- image: File (Image) [OPTIONAL if image_key given] -> Automatically uploaded to S3
- image_key: (Text) [OPTIONAL if image given]
Response (201 Created): Returns JSON.

[PUT] /api/products/{id}/
Auth Required: Yes
Content-Type: multipart/form-data
Description: Fully update an existing product.
Request Body (Send as FormData):
- name: "Classic Carrara" (Text) [REQUIRED]
- company: 1 (Number/FK) [REQUIRED]
- category: 2 (Number/FK) [REQUIRED]
- size: "800x800" (Text) [REQUIRED]
- color: "White" (Text) [REQUIRED]
- finish: "Matt" (Text) [REQUIRED]
- sku: "SKU-CAR-01" (Text) [REQUIRED, Unique]
- image: File (Image) [OPTIONAL] -> Automatically uploaded to S3
Response (200 OK): Returns updated JSON.

[PATCH] /api/products/{id}/ & [DELETE] /api/products/{id}/
Auth Required: Yes
